# Plant photos

Put local plant photos in this folder.

Recommended naming:
- lowercase
- hyphen-separated
- one primary image per plant to start

Examples:
- `lomandra-breeze.jpg`
- `ligustrum-privet.jpg`
- `pittosporum.jpg`

Then reference the file from the `image` column in:
- `admin/plant-schedule-template.csv`

Example value:
- `assets/plants/lomandra-breeze.jpg`

Notes:
- local images are safer and more reliable than web-fetched images
- app still works normally if a plant has no image
- prefer compressed JPG or WebP for faster loading
